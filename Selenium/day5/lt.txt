Application Commands
    get() - opening the application URL

    Current_url - to capture the title current url of the webpage
    Page_source - to capture source code of the webpage

Conditional Commands
    is_displayed()
    is_enabled()
    is_selected()

Browser Commands
    close() -  close single browser window
    quit() - close multiple browser windows

Navigational Commands
